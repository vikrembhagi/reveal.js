Widgets 2.0 requirements
Business and UX Requirements
Data is navigable
Aggregated and/or visualized data can be "drilled through" to show the user "raw" unaggregated data and performance values.
Analysis is explainaible
Machine Learning models, analysis and metric calculations should be explainaible to the user to instill trust in the system
Widget calculations are customizable
The benchmarks and levers of the analysis presented in the widgets should be tweakable
UI Requirements
Consistent and Predictable UI
Widget interactions and UI should feel intuitive and consistent across all widgets(as much as possible)
Modular component widgets
Each self contained widget can work independent from anything else on the page, and can have its own set of properties that are persisted based on user preferences and interactions.
Scalable
Widget container and information components should be scalable and adaptive to user screen and data variability.
